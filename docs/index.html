Hakkımda

Ben Kimim?

Merhaba, ben Berkan Berker. Bilgisayar Mühendisliği lisans eğitimimi başarıyla tamamlayarak yazılım dünyasına profesyonel bir adım attım. Eğitim hayatım ve bireysel çalışmalarım boyunca odağımı ağırlıklı olarak .NET ekosistemine çevirdim; C#, ASP.NET Core ve MVC mimarisi kullanarak ölçeklenebilir, güvenli ve modern web uygulamaları geliştirme konusunda yetkinlik kazandım.

Teorik bilgilerimi pratiğe dökmek adına MS SQL veritabanı yönetimi, Entity Framework ve RESTful API entegrasyonları içeren çeşitli projelerde aktif rol aldım. Backend geliştirme tutkumu sadece web teknolojileriyle sınırlı tutmayıp, Python ile veri analizi ve yapay zeka entegrasyonları üzerine de çalışmalar yürütüyorum.

Ayrıca Siber Vatan programı kapsamında 1 yıl boyunca beyaz şapkalı hacker teknikleri, Kali Linux, web uygulama güvenliği, tersine mühendislik ve sızma testi metodolojileri üzerine kapsamlı eğitimler aldım. Teknolojiyi bir araçtan ziyade bir problem çözme sanatı olarak görüyorum.
